# Анимации (Manim)

Оваа папка содржи анимирани математички видеа креирани со Manim.

## Структура
- Организирани по класови (`grade_06/`, `grade_07/`, итн.)
- Секое видео има `.py` датотека со Manim код

## Користење
```bash
manim -pql име_на_датотека.py ИмеНаСцената
```

## Инсталација на Manim
```bash
pip install manim
```
